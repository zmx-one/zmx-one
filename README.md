# ZMX ONE

ZMX One is a digital finance and exchange platform developed by
Zentra Mosaic Private Limited.

## Overview
ZMX One focuses on secure USDT exchange, multi-currency conversion,
wallet systems, and scalable fintech architecture.

## Key Features
- USDT Deposit & Exchange
- ZMX Internal Coin System
- Manual Withdrawals (Test Mode)
- Multi-Currency Conversion (Phase-based)
- Team & Referral System
- Bank Account Manager
- Transaction Password Security

## Status
This project is currently in early development and testing phase.
Automation and live gateways will be enabled in later phases.
